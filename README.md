# Zip Connect - Open Source HTML5 Web Game Source Code

Zip Connect game is an open source HTML5 browser game built using JavaScript.  
This project provides complete web game source code intended for developers, students, and platforms looking for lightweight and customizable browser games.

The game runs entirely in modern web browsers without requiring installation or backend services.

## Running the Project Locally

### Direct Method
Open the following file in any modern browser:

```
index.html
```

### Local Server (Recommended)

```bash
npx serve .
```

Open the generated localhost address in your browser.
